## Objetivo

O objetivo desse aplicativo é auxiliar pessoas com dificuldades de realizar tarefas com o incentivo de amigos, através de notificações criadas pelo próprio usuário ou por um de seus ***amigos***.


## Foco

o foco seria para **pessoas com dificuldade de lembrar** de realizar as suas tarefas, pessoas com **TDAH** que desejam ter auxilio para lembrar de realizar tarefas ou se lembrar de algo.
